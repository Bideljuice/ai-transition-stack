# Content Capture Quality Checklist

## Before delivering anything, verify:

### Source & Insight
- [ ] Core insight is ONE sentence — specific, not vague
- [ ] Insight is explained in simple terms Tam understands
- [ ] Tam has confirmed understanding before post is drafted
- [ ] Angle is contrarian or unexpected — not the obvious take

### Audience Check
- [ ] Content speaks to someone navigating job/financial 
      security in the AI revolution
- [ ] Assumes reader is smart but not technical
- [ ] Avoids talking down or over-explaining

### Format-Specific Checks

LinkedIn Post
- [ ] Hook is a statement, not a setup or question
- [ ] One idea only — no tangents
- [ ] No paragraph longer than 3 sentences
- [ ] Can be read in 60 seconds

Twitter Thread
- [ ] First tweet works as a standalone hook
- [ ] Each tweet is one idea — no run-ons
- [ ] Thread builds — each tweet earns the next
- [ ] Final tweet lands the point, doesn't trail off

Single Tweet
- [ ] Under 280 characters
- [ ] No filler — every word earns its place
- [ ] Punchy enough to stop the scroll

Long-form Article
- [ ] Has a clear argument — not just a topic
- [ ] 3 supporting points identified
- [ ] Data or examples noted where needed

Email Newsletter
- [ ] Subject line is specific and curiosity-driven
- [ ] One main story — not a roundup
- [ ] Written to one person, not a crowd

### Voice & Output
- [ ] Written in Tam's voice — casual, sharp, story-driven
- [ ] No meta-commentary at the end
- [ ] No filler sentences
- [ ] CTA is direct action or genuine reflection question
- [ ] No generic CTAs ("follow for more" rejected)

## AI Writing Self-Check — CRITICAL
Claude must check every output against these before delivering.
If any box is unchecked — rewrite before delivering.

### Structural
- [ ] No unnecessary headers or sections
- [ ] No random bolding or italics for emphasis
- [ ] No bullet points inside bullet points
- [ ] Formatted like a human wrote it — 
      not like a document was generated

### Language
- [ ] No "Honestly" or "Actually" dropped mid-sentence
- [ ] No tricolon over-qualification: 
      "it's not just X, it's also Y, and even Z"
- [ ] No corporate "just noticed" framing
- [ ] No self-referential language
- [ ] No balanced "on one hand / on the other hand" structure —
      takes a clear position

### The Core Test
- [ ] Could this have been written by anyone? 
      If yes — rewrite
- [ ] Does it sound like Tam specifically?
      Casual, sharp, story-driven, no fluff
- [ ] Is there at least one line that could 
      standalone as a single tweet?
- [ ] Does it remove an excuse OR reframe 
      something familiar unexpectedly?
